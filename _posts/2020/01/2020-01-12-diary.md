ひたすら体論の章末問題をこなす。最小多項式の決定問題辺りでもたもたしている。
多項式自体は求めるが、それが本当に最小多項式なのかを証明する方法がわからないまま。
十分条件しか知らない。

あと SymPy の `root()` で $n$ 乗根の $k$ 番目を得られることがわかった。
しかし計算に難がある。

```python
>>> from sympy import *
>>> omega = root(1, 3, 1); omega
(-1)**(2/3)
>>> re(omega), im(omega)
(-1/2, sqrt(3)/2)
>>> omega**2
-(-1)**(1/3)
>>> re(_), im(_)
(-1/2, -sqrt(3)/2)
>>> re(omega**3), im(omega**3)
(1, 0)
>>> omega**2 + omega + 1
1 - (-1)**(1/3) + (-1)**(2/3)
>>> simplify(_)
1 - (-1)**(1/3) + (-1)**(2/3)
>>> expand(_)
1 - (-1)**(1/3) + (-1)**(2/3)
```
