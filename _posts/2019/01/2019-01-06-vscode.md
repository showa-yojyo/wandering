title: VS Code スニペットの scope についての利用メモ
tags: vscode
---

コードスニペット構成スクリプトでは `scope` を指定することでコード片展開を特定のメジャーモードに限定できる。

{% raw %}
```json
{
    "PostUrl": {
        "scope": "markdown",
        "prefix": "post_url",
        "body": "[$1]({{ site.baseurl }}{% post_url \$2 %})",
        "description": "post_url directive"
    }
}
```
{% endraw %}

また、プレースホルダーで選択肢を提示できる。
